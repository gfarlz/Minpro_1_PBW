# Website Portofolio

## Deskripsi Project
Website portofolio sederhana berbasis HTML dan CSS dengan bantuan Bootstrap 5. Website ini terdiri dari tiga section utama yaitu Home, About Me, dan Certificates. Website bersifat statis dan responsif.

---

## Tampilan Setiap Section

### 1. Home Section
Berisi:
- Nama
- Perkenalan singkat
- Foto profil
- Navbar

Home menggunakan Bootstrap untuk layout dan CSS custom untuk styling gradient serta tampilan visual.

---

### 2. About Me Section
Berisi:
- Deskripsi diri
- Pengalaman
- Skills menggunakan progress bar

Progress bar dibuat menggunakan komponen Bootstrap dan dimodifikasi dengan CSS untuk memberikan tampilan yang lebih menarik.

---

### 3. Certificates Section
Berisi:
- Daftar sertifikat dalam bentuk card
- Layout grid menggunakan Bootstrap (row & col)
- Gambar sertifikat dengan rasio 16:9

Card dibuat menggunakan komponen Bootstrap dan disusun responsif menggunakan Grid System.

---

## Penjelasan Code

### Struktur HTML
- Menggunakan struktur dasar HTML (html, head, body)
- Menggunakan semantic section untuk setiap bagian
- Navbar menggunakan komponen Bootstrap

### Styling CSS
- Menggunakan file eksternal style.css
- Mengatur warna, background, font, spacing
- Mengatur rasio gambar sertifikat menggunakan aspect-ratio 16:9
- Custom styling untuk progress bar dan card

### Responsive Design
Menggunakan:
- Bootstrap Grid System (container, row, col-md-4)
- Breakpoints bawaan Bootstrap
- Media query tambahan pada CSS

---

## Teknologi yang Digunakan

- HTML5
- CSS3
- Bootstrap 5.3.8

---

## Kesimpulan
Website portofolio ini dibuat untuk memenuhi tugas praktikum dengan menerapkan struktur HTML yang benar, styling menggunakan CSS, serta memanfaatkan Bootstrap 5 untuk membantu proses layouting dan responsive design.
